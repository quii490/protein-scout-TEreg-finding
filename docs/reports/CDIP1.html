4. 总体评价

CDIP1 是一个 p53 靶基因编码的小型内体/溶酶体膜蛋白，功能明确指向凋亡和内体分选。核定位证据薄弱（仅 GO IDA:MGI 鼠源推断，UniProt Subcellular Location 完全不支持核定位）。PPI 网络丰富但均为内体/泛素通路相关。归一化总分 58.5/100。不建议作为核-胞质蛋白候选——其功能定位在膜系统，核定位可能为假阳性或间接现象。
