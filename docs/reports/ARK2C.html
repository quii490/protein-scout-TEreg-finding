4. 总体评价

ARK2C 是一个文献量极低（PubMed strict=3）的 E3 泛素连接酶，在 BMP-Smad 信号中调控运动神经元轴突延伸。核定位仅有序列类比/IBA 预测证据，未达到实验确认水平。8 个 PDB 结构仅覆盖 C 端 RING domain（255-346），N 端无序区 pLDDT 较低。作为 nucleoplasm 候选置信度低，建议补内源 IF 确认核定位后重新评估。
