4. 总体评价

PHF19 是 PRC2 复合体的关键辅助因子，介导 H3K36me3 识别到 H3K27me3 沉积的组蛋白修饰转换。核定位有 UniProt curated 支持但缺 HPA IF 验证。结构域覆盖良好（PHD + Tudor），PPI 网络以 PRC2 为核心极为稳固。研究热度中等偏高 (strict=62)，在 Polycomb/染色质调控领域 niche 明确。
