# GitHub Pages Setup

This repository is prepared for a static GitHub Pages site generated under `docs/`.

After the repository is uploaded to GitHub:

1. Open Repository Settings.
2. Open Pages.
3. Set Source to `Deploy from a branch`.
4. Set Branch to `main`.
5. Set Folder to `/docs`.
6. Save.
7. The entry page is `docs/index.html`.

Notes:

- Step 4B only builds the local `docs/` site.
- GitHub repository creation, upload, and Pages activation are reserved for later Step 5/6.
- Do not claim the site is online until GitHub Pages is explicitly enabled and verified.
